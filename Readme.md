# Analizador de documentos PDF con IA

Sube cualquier PDF y hazle preguntas en lenguaje natural. 
La app usa GPT para analizar el contenido y responder.

## Tecnologías
- Python
- Streamlit
- OpenAI GPT-4o-mini
- PyPDF2

## Instalación
pip install -r requirements.txt

## Uso
1. Añade tu API key de OpenAI en un archivo .env
2. Ejecuta: streamlit run app.py
3. Sube un PDF y empieza a preguntar
